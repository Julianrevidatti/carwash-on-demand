## 📸 Capturas de Pantalla

Para agregar capturas al sistema:

1. Navega a `http://localhost:3000`
2. Toma screenshots de cada módulo
3. Guárdalas en `/public/tutorials/screenshots/`
4. Nombra según el comentario en cada guía: `<!-- SCREENSHOT: nombre_archivo.png -->`
